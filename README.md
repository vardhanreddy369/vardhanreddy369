### Who I Am

I'm the kind of engineer who can't just use a tool — I have to build one.

My biggest project, **SubTrack**, processes **1.19M+ corporate subsidiary records** across 22K+ public companies. I built the entire thing: async data pipeline, AI enrichment engine (Google Gemini + SEC EDGAR + Wikipedia), 3D network visualizations, real-time streaming search, Stripe billing. The classification system hits **94% accuracy** and I'm currently pushing that higher with XGBoost + FinBERT.

**What makes me different:** I sit at the intersection of AI/ML and full-stack engineering. I can train a model in the morning, build the API to serve it in the afternoon, and push the frontend that displays it by evening. Most people specialize in one lane — I connect all three.

Every project below has a **live demo**. I don't do README-only projects.
